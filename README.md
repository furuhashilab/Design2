#  ワールドカップの歴代優勝国×観光地に関する3Dストーリーテリングマップ

## 公開URL 
https://human-silver-c33ulk6a.edgeone.dev/

## 概要
- **観光地やその国の優勝回数、有名選手**をテーマとしたストーリーテリング
- 単なる静止画としての観光地説明ではなく、国と国をジャンプして巡るような移動がある
- 優勝回数については、何回という数字のみならず、**具体的な年次**についても記載されている

---

## 主な特徴

- 観光地やその説明のみならず、一目で様々な情報を知ることができる
- 自動で各国を飛び回り、閲覧者は楽しみながら学ぶことができる
- **ワールドカップにや選手にまつわる文章**も登場する

---

## 参考(要件)
- ﻿﻿使用したライブラリ： MapLibre GL Js
- ﻿﻿ベースマップ： OpenEreeMap（ https://tiles.openfreemap.org/styles/1iberty ）
- ﻿﻿3D表現： fi11-extrusionを使用し、ビルディングデータを立体的に表示
- ﻿﻿カメラワーク：
- ﻿直下視（pitch 0）は禁止。常にpitch 45度～60度を維持すること。
- ﻿﻿シーンごとに座標（緯度経度）へ移動し、自動的にポップアップを表示すること。
- ﻿﻿カメラはシーンごとに旋回や位置調整を行い、ダイナミックに演出すること。
- ﻿﻿公開先：本ページ(GitHub Pages)
