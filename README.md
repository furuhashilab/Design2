#  ワールドカップの歴代優勝国×観光地に関する3Dストーリーテリングマップ

## 公開URL 
https://human-silver-c33ulk6a.edgeone.dev/

## 概要
私たちのプロジェクトでは、**観光地やその国の優勝回数、有名選手**をテーマとしたストーリーテリングを作成しました。単なる静止画としての観光地説明ではなく、国と国をジャンプして巡るような自動移動があり、所々にワールドカップとの交わりについての説明もある
優勝回数については、何回という数字のみならず、優勝年次についても記載されている

---

## 主な特徴

- 代表的な観光地名や観光地の説明飲みならず、優勝回数と有名な選手についても知ることができる
- 自動で各国を飛び回り、楽しみながら学べる
- ワールドカップにや選手にまつわる説明も登場する
